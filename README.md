# Frozen Lake


## Requirements
create a .env with the openAI key and python >=3.10.12


## Install Dependencies

```sh
  python3 -m venv .venv
  source .venv/bin/activate
  pip3 install -r requirements.txt
```

## Run locally 
```sh
  python3 vi_and_pi main.py
```
